# Business Modules Roadmap

This backend scaffold already includes:

- Auth (JWT login)
- Users list by tenant
- Attendance check-in with geofence check
- Tasks create + list my tasks
- Realtime Socket.io gateway

Add these modules next (same structure pattern):

- `manager/` roster planning, swap approval, stock and density logs
- `accounting/` cash report verification and denomination audit
- `dsm/` daily collections and swap request submission
- `cleaning/` task-only role workflows and SLA tracking
- `notifications/` push and in-app delivery channels
- `reports/` dashboard aggregates for attendance, revenue, completion
